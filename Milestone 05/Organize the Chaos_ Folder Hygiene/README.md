# ShopFlat — feature-based folder structure

E-commerce demo (auth, products, cart, orders) refactored from a flat `src/` layout into `features/`, `components/`, `hooks/`, `utils/`, and `services/`.

## Run locally

```bash
npm install
npm run dev
```
